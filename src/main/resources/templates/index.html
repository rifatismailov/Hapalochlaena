Description of the project

Hapalochlaena — це система для виявлення критичної інформації у текстових документах. Вона аналізує текст на основі семантичної схожості з заздалегідь визначеними шаблонами (темплейтами), що описують ознаки критичності.

Суть роботи полягає в наступному:

  • Шаблони мають ключові поля (наприклад, мета, підстава, наказ), кожне з яких перетворюється в ембеддінг-вектори.
  • Отриманий документ розбивається на рядки, і для кожного рядка генеруються вектори значення.
  • Система порівнює вектори документа з шаблонами й обчислює cosine similarity.
  • Якщо схожість перевищує поріг, система визначає, що текст містить потенційно критичну інформацію.

Дані можуть надходити з різних джерел (локальні файли, API, веб тощо), що дозволяє системі діяти як “щупальця” — гнучко охоплювати всі можливі канали.
